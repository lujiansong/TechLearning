# Python Development Environment Setup - Quick Start Guide

Essential steps to set up Python development on Windows with VS Code.

## 1. Install Python

**Download & Install:**
- Visit [python.org/downloads](https://www.python.org/downloads/)
- Download Python 3.11+ 
- **Important:** Check "Add Python to PATH" during installation

**Verify:**
```powershell
python --version
pip --version
```

## 2. Install VS Code

- Download from [code.visualstudio.com](https://code.visualstudio.com/)
- Install with default settings

## 3. Essential Extensions

Install via Extensions panel (`Ctrl+Shift+X`):
- **Python** (Microsoft) - Main Python support
- **Pylance** (Microsoft) - Language server
- **Python Debugger** (Microsoft) - Debugging support

## 4. Configure Python Environment

**Select Interpreter:**
- Press `Ctrl+Shift+P`
- Type "Python: Select Interpreter"
- Choose your Python installation

**Test Setup:**
Create `test.py`:
```python
print("Hello, Python!")
```
Run with `F5`

## 5. Virtual Environment (Recommended)

**Create virtual environment:**
```powershell
# Navigate to project folder
cd "C:\YourProject"

# Create virtual environment
python -m venv venv

# Activate
.\venv\Scripts\Activate.ps1
```

**Configure VS Code:**
- `Ctrl+Shift+P` ‚Üí "Python: Select Interpreter"
- Select interpreter from `venv` folder

## 6. Basic VS Code Settings

Create `.vscode/settings.json`:
```json
{
    "python.defaultInterpreterPath": "./venv/Scripts/python.exe",
    "python.terminal.activateEnvironment": true,
    "editor.formatOnSave": true,
    "python.formatting.provider": "black"
}
```

## 7. Package Management

**Install packages:**
```powershell
pip install requests numpy pandas black pylint
```

**Save requirements:**
```powershell
pip freeze > requirements.txt
```

## 8. Quick Debugging

**Set breakpoints:** Click left margin next to line numbers
**Debug shortcuts:**
- `F5`: Start debugging
- `F9`: Toggle breakpoint
- `F10`: Step over
- `F11`: Step into

## 9. Code Quality Tools

**Install:**
```powershell
pip install black pylint
```

**Format code:** `Shift+Alt+F`
**Enable linting:** Settings ‚Üí Python ‚Üí Linting ‚Üí Enable

## 10. Git Integration (Optional)

**Install Git:**
- Download from [git-scm.com](https://git-scm.com/)
- Configure:
```powershell
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```

**Use in VS Code:**
- Source Control panel: `Ctrl+Shift+G`
- Stage files with `+` button
- Commit with `Ctrl+Enter`

## Essential Keyboard Shortcuts

- `Ctrl+Shift+P`: Command palette
- `Ctrl+`` `: Toggle terminal
- `F5`: Run/Debug
- `Ctrl+/`: Toggle comment
- `F12`: Go to definition
- `Shift+Alt+F`: Format document

## Quick Troubleshooting

**Python not found:** Reinstall Python with "Add to PATH" checked
**Module not found:** Activate virtual environment, install packages
**Debugger not working:** Verify correct interpreter is selected
