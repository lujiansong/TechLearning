## 2. 数据类型
- **数字**：`int`, `float`
- **字符串**：`str` - 单引号或者双引号都可以
- **布尔值**：`True`, `False`
- **集合类型**：
  - 列表（有序，可变）：`[1, 2, 3]`
    元素之间没有任何关系， Index从0开始，-1是倒数第一个， -2...; list[start:end]： start：起始索引（包含），end：结束索引（不包含），如果省略 start，默认从头开始；省略 end，默认到末尾。
  - 元组（有序，不可变）：`(1, 2, 3)` 不能修改其中元素，但是可以直接变量赋值
  ```
  dimessions = (200,150)
  dimessions[0]=100            -x
  dimessions=(100,100)
  ```
  - 字典（键值对）：`{"key": "value"}`
  - 集合（不重复元素）：`{1, 2, 3}`


示例：
```python
n = 42        # int
x = 3.14      # float
name = "Jesse"  # str
ok = True     # bool
nums = [1,2,3]
point = (10, 20)
user = {"id": 1, "name": "Jesse"}
unique = {1,2,2,3}  # => {1,2,3}
```

## 3. 变量与运算
- 变量名区分大小写。 建议用小写字母，并遵循PEP 8命名规范。
    - 小写字母 + 下划线分割
    - 不要使用大写开头（大写通常用于类名）
    - 避免使用保留字（如 class, def）
- **类名**： 是同驼峰命名法(PascalCase) `calss StudentInfo:`
- 常量：全部大写，单词间用下划线 `MAX_SIZE =`

```python
x = 10
y = 3
print(x + y)   # 加法 13
print(x - y)   # 减法 7
print(x * y)   # 乘法 30
print(x / y)   # 浮点除法 3.333...
print(x // y)  # 整除 3
print(x % y)   # 取余 1
print(x ** y)  # 幂 1000
```

## 5. 循环
```python
# for 循环
for i in range(5):
    print(i)  # 0,1,2,3,4

# while 循环

```

## 8. 类与对象（OOP）
```python
class Dog:
    def __init__(self, name):
        self.name = name  # 实例属性

    def bark(self):
        print(f"{self.name} says Woof!")

# 使用类
pet = Dog("Buddy")
pet.bark()

# 继承
class Animal():
    def __init__(self,name, age):
        self.name = name
        self.age = age
    def sit(self):
        print(self.name.title() + " is sitting")
class Dog(Animal):
    def __init__(self, name, age, color):
        super().__init__(name,age)
        self.color = "Green"
子类同名函数覆盖父类同名函数

```
**概念**：封装、继承、多态。常用魔术方法：`__init__`, `__str__`, `__repr__`。
